# 💰 工资计算器

纯前端工资计算器，宜昌社保 + 排班轮换 + 节假日动态获取，零后端依赖。

## 技术栈

Vite + TypeScript (strict) · 纯原生 HTML/CSS · dayjs · china-holiday · zod

## 快速开始

```bash
pnpm install
pnpm dev      # 本地开发
pnpm build    # 打包 → dist/，可直接部署 CF Pages / Vercel / Netlify
```

## 功能

| 模块 | 说明 |
|------|------|
| **单月计算** | 选年月 + 排班配置 → 实时出薪资明细（加班/社保/个税/到手） |
| **多月批量** | 起止年月 → 分页表格 + 汇总（总税前/社保/个税/到手/月均） |
| **一键复制** | 单月明细 / 多月汇总一键复制为文本 |

## 排班体系

| 班型 | 含义 | 工时 | 倍率 |
|------|------|------|------|
| A班 | 普通工作日 | 8h + 3h 加班 | 加班 ×1.5 |
| B班 | 休息日前一天 | 11h | ×2 |
| C班 | 固定休息日 | 休息 | — |
| F班 | 法定节假日 | 11h | ×3 |

- C班可自选周几，B班自动为前一天
- 支持按日期 / 按周几设置不加班（仅出勤 8h）
- 隔月白/夜班轮换，夜班补贴 20 元/天
- F班冲突：节假日撞 B/C 班时临时后移，仅当周有效

## 社保 & 个税

- 宜昌社保基数 4299：养老 8% · 医疗 2% · 失业 0.3% · 大额+长护 14.95/月
- 个税起征点 5000，税率 3%，无专项附加扣除
- 计税基数 ≤ 0 时免征
